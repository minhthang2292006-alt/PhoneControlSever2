PhoneControlServer - Điều khiển PC bằng điện thoại qua Web

Cách chạy:
1. Cài .NET 8 SDK (hoặc đổi TargetFramework phù hợp nếu dùng .NET 6/7).
2. Mở terminal tại thư mục project.
3. Chạy lệnh: dotnet run
4. Lấy IP máy tính trong cùng Wifi/LAN.
5. Trên điện thoại mở: http://IP_MAY_TINH:5000 ( cmd -ipconfig- xuat hien ip của pc hoac vào task manager )
ip của pc
port:5000
Lưu ý:
- Chạy trên Windows.
- Có thể cần mở firewall cho cổng 5000.
